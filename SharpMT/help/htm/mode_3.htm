Preview tab
The Preview tab uses an embedded browser control to show you what your current text looks like when properly rendered, based on the previewHtml.htm file. While CSS sheets are not available in this preview, the CSS style can be embeded within the previewHtml.htm file using the <style> tag. Also, if this draft's Text Formatting is set to "Convert Line Breaks" then its effects will be reflected in this preview; this is the only Text Formatting option that will effect SharpMT.

Preview View Options
