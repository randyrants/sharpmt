Main Menu

File
NameShortcutCommand
New DraftCtrl+NCreates a new draft
Open Draft...Ctrl+OOpens an existing draft
Close DraftCtrl+F4Closes the current draft
Save DraftCtrl+SSaves the current draft
Save All Drafts Saves all open drafts
(MRU List)List of the eight most recent opened drafts
ExitAlt+F4Closes SharpMT

Edit
NameShortcutCommand
CutCtrl+XCuts text from current text box
CopyCtrl+CCopies text from current text box
PasteCtrl+VPastes text into current text box
DeleteDELDeletes either text or current Blog Link, depending on current focus
Select AllCtrl+ASelects all text in the current text box
Find...Ctrl+FOpens the Find dialog box

Format
NameShortcutCommand
BoldCtrl+BWraps the current selected text (if any) with Bold tags
ItalicCtrl+IWraps the current selected text (if any) with Italic tags
UnderlineCtrl+UWraps the current selected text (if any) with Underline tags
URL...Ctrl+LOpens the Add URL dialog that will insert a new link
Now PlayingCtrl+MInserts information provided by the current MP3 player
Custom Tags Shows a popup menu with a list of custom tags, as defined in the Tags Options (if any)

Tools
NameShortcutCommand
Spell CheckF7Spell Check the current draft manually
Spell Check Options... Opens the Spell Check Options dialog
Post to ServerCtrl+Shift+PPosts the current draft to the server
Post All Open Drafts Posts all open drafts to the server
Update Blog Links Gets an updated list of Blog Links from the server
Update Categories Gets an updated list of Categories from the server
Update Text Filters Gets an updated list of Text Filters from the server
Upload Image... Opens the Upload Image dialog
Options... Opens the Options dialog

Window
NameShortcutCommand
Blog Links Shows or Hides the Blog Links window
Main ModeCtrl+EView the main fields for the current draft
Advanced ModeCtrl+DView the advanced fields for the current draft
Preview ModeCtrl+PView the preview mode for the current draft
Next DraftCtrl+TabSwitches to the next open draft
Previous DraftCtrl+Shift+TabSwitches to the previous open draft

Help
NameShortcutCommand
HelpF1Shows the Help system
FAQCtrl+F1Shows the frequently asked questions document
Tool Tips Shows or hides tooltips for most controls
Check for Updates Manually check for updates - this will require an Internet connection
About... Information about this version of SharpMT

not all commands are available at all times
