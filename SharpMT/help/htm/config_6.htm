Proxy Options
Options in this panel are used to set up a proxy server, if you require one, when accessing the network.

Proxy Options
Use a proxy for HTTP command:
If you know that you need a proxy server to access the network, check this box and fill out the rest of these options
Proxy Server Name:
Enter the network name of your proxy server. For example: www.proxyname.com http:// is not required for this field
Proxy Port:
Enter the number of the proxy port that your server talks to.

Proxy Server Options
My Proxy requires authentication
If your proxy server requires a user name and password, check this box and fill out the options below.
User Name:
Enter the user name that you use to access your proxy server.
Password:
Enter the password that you use to access your proxy server.
