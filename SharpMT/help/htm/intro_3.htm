First Time Users

Getting a Server
For this task, you're on your own - all of the following steps assume that you have a server and valid account on that server already!

Configuring SharpMT
The first and most important configuration step of SharpMT is getting it connected to a Blog server. Without that, there's not going to be too much that you can do with SharpMT as most of its features are dependent on the Blog server.

    Accessing SharpMT Options:
  • Click on the Tools menu and select "Options..."
  • Across the top of this dialog there will be different panels to view; select Server

Server Information:
Web Server Name:
Enter the Internet address or network name of your Blog server. For example: www.hostname.com http:// is not required for this field --- for TypePad users it is always www.typepad.com
CGI-BIN Path:
Enter the path and file name to your XML-RPC module. For MT users, this file is named mt-xmlrpc.cgi and will be located in your CGI-BIN directory. For example, if your link to access your server-based Blog settings is http://www.hostname.com/cgi-bin/mt.cgi then your CGI-BIN path is: /cgi-bin/mt-xmlrpc.cgi --- for TypePad users it is always /t/api
User Name:
Enter the user name that you use to access your Blog's settings.
Password:
Enter the password that you use to access your Blog's settings.
Blog List:
If you click Refresh now, you will see a list of your Blogs appear in this list.
Refresh
Press this button to refresh your Blog-based settings from your server. You should refresh whenever you've changed your server name, Blog published, Categories, Text Filters, or anything else that is defined by the server.

When you click OK on this dialog, SharpMT will automatically download a list of Categories and Text Filters from your Blog server as those are also required to effectively work offline and when posting new drafts. Once that is completed you are now ready to Blog writing while offline.
