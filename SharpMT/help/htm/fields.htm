SharpMT Fields List

Image Tag Options
Set via the Options dialog on the Tags panel.
Field nameText inserted
[IMAGELINK]The URL to the newly uploaded image
[ALTTAG]The ALT text that was entered on the Upload Image dialog

Now Playing/Music Options
Set via the Options dialog on the Editor panel.
Field nameText inserted
[ALBUM]Album/CD title
[ARTIST]Recording artist's name
[DURATION]Duration of current song
[SONG]Song title

Preview View Options
Set in the previewHtml.htm file, found in the SharpMT directory.
Field nameText inserted
<$MTEntryBody$>The text from the Entry Body field
<$MTEntryCategory$>The currently select primary category
<$MTEntryExcerpt$>The text from the Excerpt field
<$MTEntryMore$>The text from the Extended Entry field
<$MTEntryTitle$>The title that is set in the Title field

RSS Aggregator Options
Set in the RSSFormat.txt file, found in the users' Application Data\RandyRants.com\SharpMT directory. Quite often this would be C:\Documents and Settings\YourUserName\Application Data\RandyRants.com\SharpMT\RSSFormat.txt
Field nameText inserted
[RSSDESCRIPTION]The RSS' story description (if available)
[RSSLINK]The RSS' story link
[RSSTITLE]The RSS' story title
