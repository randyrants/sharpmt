Installation
If you're reading this help file, you've probably gotten SharpMT installed already. The installation will not run without the required version of the .NET Framework, so that too is a non-concern at this point.

Uninstall Information
  • Open your Control Panel and run the Add or Remove Programs application.
  • Select SharpMT and click Remove to start the Uninstall Wizard.
  • Follow the instructions of the Uninstall Wizard to complete the removal of SharpMT.
